# py_axieGenes
# Python Axie Infinity genes parser

# Initial resources:
 - Parse axie's class, parts d, r1 and r2 by giving the genes hex string (default to 512 bits newGenes). Parts mapping complete with specialGenes, stage 2, type, class and id.
